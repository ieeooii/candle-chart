## 기술 스택 (결정됨)

- [x] **Vite + React + TypeScript** (`pnpm create vite --template react-ts`) — TS·React는 자격요건, Vite는 우대사항
- [x] pnpm 사용
- **그리기**: Canvas 2D (브라우저 내장 API, 설치 불필요) — 캔들차트 본체는 WebGL 아님
- **데이터**: 업비트 시세 API (계정·인증 불필요)

## 데이터 소스 (A 방식: 데이터 먼저)

- REST 과거 캔들: `GET https://api.upbit.com/v1/candles/seconds?market=KRW-BTC&count=200`
  - 문서: https://docs.upbit.com/kr/reference/list-candles-seconds
- WebSocket 실시간 캔들: 구독 메시지 `[{ticket}, {type:"candle.1s", codes:["KRW-BTC"]}, {format:"DEFAULT"}]`
  - 문서: https://docs.upbit.com/kr/reference/websocket-candle
- 쓸 필드(OHLC): `opening_price`(시가) / `high_price`(고가) / `low_price`(저가) / `trade_price`(종가)

> ⚠️ **주의사항 메모**
>
> - REST 응답은 최신순(내림차순) → 왼→오 시간순으로 그리려면 `reverse()` 필요할 수 있음
> - REST `fetch`가 CORS로 막히면 Vite 프록시로 우회 (막히면 그때 처리)
> - WebSocket 응답이 Blob(바이너리)로 올 수 있음 → 텍스트 변환 후 `JSON.parse`
> - `codes`는 반드시 대문자 (`KRW-BTC`)
